# pngtojpgconverter
📄✨ PDF → JPG Converter
A lightweight and user-friendly Python tool to convert PDF pages into high-quality JPG images.
Whether you’re extracting slides from a presentation, saving book pages as images, or just need a quick preview of a PDF — this tool gets the job done in seconds 🚀.


🔹 Features
✅ Fast Conversion – Converts PDF pages to JPG instantly
✅ High Quality Output – Maintains sharp text and images
✅ Multi-page Support – Extract all pages or select specific ones
✅ Easy-to-use GUI – No coding needed, just click and convert
✅ Lightweight & Portable – Runs without heavy dependencies


🛠 Tech Stack
Python 🐍
Tkinter 🎨 – Simple and clean graphical interface
pdf2image / PyMuPDF 📄 → 📷 – PDF rendering to images
Pillow (PIL) 🖼 – Image processing & saving


📦 Installation
Make sure you have Python 3.x installed. Then install the dependencies:
pip install pillow pdf2image


🚀 Usage
Run the Program
python pdftojpg_converter.py


1️⃣ Select your PDF from the file dialog 📂
2️⃣ Choose where to save the images 💾
3️⃣ Click Convert and let the magic happen ✨
4️⃣ Your PDF pages are now high-quality JPG images 🎉


📂 Output
Naming format: page_1.jpg, page_2.jpg, ...
Saved in your chosen folder
Supports custom DPI settings for better resolution


🎯 Use Cases
Extracting presentation slides from PDFs
Saving book/magazine pages as images
Making social media posts from PDF designs
Quick preview for PDFs without a reader
